Welcome to Card Match!

Match all the cards to win. Good luck!

This is a simple card match game, which was created as a technical appraisal for Entain. The game uses Pixi.js for graphics, and Howler.js for simple integration of sound effects/music. It was written in WebStorm but, due to problems with Pixi, should be run with Vite. Please excuse the dreadful AI generated title screen. I am NO artist. The packed version is available here. and the Github repo is here.
